Formatando STRINGS

var s = ' JavaScript'

  • ' ou "Eu estou aprendendo s" ou ' -> Não faz interpolação
  • 'Eu estou aprendendo' + s -> Usa concatenação
  • `Estou aprendendo ${s}` -> usa template string

Utilizando o + para somas/adição
  • É preciso estar [number+number]
  • Porém, WINDOW.PROPT ele só recebe um valor string, portanto, converter o string para number!

    • Number()

    Para transformar o [string -> number], devemos utilizar o NUMBER, e o próprio javascript decide se a resposta será número inteiro ou número real, porém, podemos definir regras para ser apenas uma das respostas, abaixo

    Number.parseInt(n) ou Number.parseFloat(n)

    Para utilizar apenas números inteiros
  • Number.parseInt( dentro dos parentesis, utilizar a funcao desejada, neste caso window.prompt)
    • Para utilizar números reais (com casas decimais)
  • Number.parseFloat (dentro dos parentesis, utilizar a função desejada, neste caso, window.prompt)

    • Conversão de number para string
