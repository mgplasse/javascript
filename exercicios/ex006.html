Aula 09 - D.O.M

What is DOM?

DOM is a abbreviation for "Document Object Model"

DOM só roda dentro do JS, quando estiver no HTML.

Você deve aprender a fazer a sua árvore DOM (árvore genealógica do window)

  • window
    • location
    • history
    • document
      • html
        • head
          • meta
          • title
        • body
          • h1
          • h2
          • p(0)
          • p(1)
          • div

Meios de selecionar:

Por marca[TAG] -> getElementsByTagName()
Por ID -> getElementById() // note que essa nao tem o S em Element, pois nao há mais de um ID igual
Por nome -> getElementsByClassName()
Por classe -> getElementsBy
